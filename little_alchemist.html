Little Alchemist


Github

The final group project for the CST426 - Advanced Game Development course. We used the Unity Engine to create a virtual science lab with three lab missions. The first lab experiments with changing the colors of a flame. The second lab experiments with the visual effects of an explosion of a balloon after filling it with hydrogen and/or oxygen. The third lab is a molecule builder, which visually represents certain molecular structures based on the atoms placed in the builder.

My main role in this project was in programming the UI elements.
