Projects


Seeker

I pitched the game idea that became Seeker to the CSUMB Game Developers Club, and led the development of the prototype game. Seeker is a first person shooter game where you play as an assassin. Search among the crowd for the provided target and eliminate them.


In The Fog

In The Fog is my first solo project. It is a first person shooter survival game. Left alone in a foggy forest, you must survive wave after wave of deadly zombies. Get kills, collect cash, buy equipment, survive.
